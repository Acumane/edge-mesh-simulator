## Edge Mesh Simulator 

### Setup
Install dependencies:

- `[p]npm install`
- `pip install -r modules`
- Install [Blender](https://www.blender.org/download/)
	- Ensure the executable is accessible on your PATH as `blender`

Run with: 
- `./run.sh`
